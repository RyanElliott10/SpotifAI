# SpotifAI
Handy dandy neural network that generates playlists from a given input library.

# Requirements
```
Keras==2.3.1
Keras-Applications==1.0.8
Keras-Preprocessing==1.1.0
numpy==1.17.0
spotipy==2.4.4
```

Run `pip3 install -r requirements.txt` to install all used libraries.
